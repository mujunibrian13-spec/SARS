Bishop Stuart University

Smart Activity Reminder System

An automated n8n workflow that watches scheduled student activities and fires timely Email & WhatsApp reminders — no manual follow-up required.

⏱ Runs every minute 📧 Email + WhatsApp 📊 Google Sheets backed 🟢 Active

Overview

SARS_Workflow_A_Activity_Reminder_Dispatch is a scheduled n8n automation that reads upcoming activities from a Google Sheet, works out which ones are due to start in the next 15 minutes, and notifies every enrolled student by email and WhatsApp — logging each notification and marking the activity as handled so nobody gets reminded twice.

1 min Trigger interval
0–15 Minute reminder window
2 Notification channels
12 Workflow nodes

How the Workflow Runs

Every execution follows the same pipeline, end to end:

  1. 1

    Schedule Trigger

    Fires automatically every minute to kick off a fresh check of the activity sheet.

  2. 2

    Read Activities Sheet

    Pulls every row from the Read Activities tab of the master Google Sheet.

  3. 3

    Calculate Time Until Activity

    A Code node computes minutesUntil for each activity by comparing its date/time to now.

  4. 4

    Due Now Filter

    Keeps only activities starting within 0–15 minutes that haven't already been notified.

  5. 5

    Loop Over Due Activities

    Processes each due activity one at a time, item by item.

  6. 6

    Get Student Details

    Looks up the enrolled student's contact info from the Students sheet by student_id.

  7. 7

    Format Reminder Messages

    Builds the email subject/body and a shorter WhatsApp message using the activity + student data.

  8. 8

    Send Email & WhatsApp Reminder

    Dispatches both notifications in parallel via Gmail and Twilio WhatsApp.

  9. 9

    Log Notifications

    Appends a record of each channel's delivery to the Notification Log sheet.

  10. 10

    Mark Activity as Notified

    Flags the activity row so the same reminder is never sent twice, then loops to the next item.

Data Model

The workflow reads and writes three tabs in one Google Sheet.

SheetKey ColumnsPurpose
Read Activities activity_id, student_id, activity_name, activity_date, activity_time, notified Master list of scheduled activities and their notification status.
Students student_id, full_name, email, whatsapp_no Contact directory used to resolve who to notify.
Notification Log log_id, activity_id, student_id, channel, sent_at, status Audit trail of every email/WhatsApp reminder sent.

Register

Submit directly into the same Google Sheet the workflow reads from. New students appear in the Students tab and new activities appear in the Read Activities tab, ready to be picked up by the next scheduled reminder run.
